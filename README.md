# Routine Tesi & Inglese — mini-app

Questa è una mini-app web/PWA per telefono.

## Come usarla
1. Carica la cartella su un hosting HTTPS, per esempio GitHub Pages, Netlify o Vercel.
2. Apri `index.html` dal telefono.
3. Dal browser usa “Aggiungi alla schermata Home”.
4. Apri l’app e premi “Attiva notifiche”.

Nota importante: i promemoria locali funzionano meglio se apri l’app al mattino. 
Per notifiche totalmente affidabili anche con app chiusa, importa anche il file calendario `.ics` allegato nel calendario del telefono.
